Introduction

C++ is an object-oriented programming language chosed by many to build large-scale, high performant applications. It is used when there is a need for performance or when low-level programming is important, here are some use cases:

  • Embedded software and robotics
  • Operating systems
  • Games
Hello World

To get started, here's how you would output to the console the text "Hello World!"

#include <iostream>

int main() {
  std::cout << "Hello World!";
}

// Outputs: Hello World!
Variables

C++ is a strongly typed language, meaning every variable must be decrared with its type, the reason is because each type takes up a different amount of space in memory so the compiler has to reserve the amount. It is possible to declare

int a; // Declare an integer variable
int a, b, c; // Declares multiple integer varaibles at once
bool yes; // Decrate a boolean variable

It is possible to declare a varaible without initializing it, but it will be assigned a random value and the compiler will throw an error, therefore, it is important to initialise it with a value.

int x = 5; // Declare and initialise an integer
int y = x + 3; // Declare and initialise an integer equal to a previously created variable
float z = 3.14; // Declare and initialise a float

std::string str; // Can declare a string
str = "Hello"; // Then initialise it

Using namespaces

#include <iostream>
using namespace std;

int main() {
  string str; // Can ommit the 'std::'
  str = "Hello";
}
Conditionals

Condionals are executed if the given condition is true. The syntax is similar to: if condition then statement. It can be written without braces if there is a single statement.

if (a == 4)
  cout << "a is 4";

There are many other operators such as

< // Less than
> // Greater than
<= // Less than or equal to
>= // Greater than or equal to

There are also different statements that can be executed if the first one is false

if (a == 4) {
  cout << "a is 4";
}

else if (a > 4) {
  cout << "a is greater than 4";
}

else {
  cout << "a is less than 4";
}
Loops

While loops repeat the given statement while it is true. The loop ends once it is no longer true. They must also have a decrement or an increment otherwise, it will be an infinite loop.

int n = 5;
while (n >= 0) {
  cout << n << endl;
  n--;
}

// Outputs numbers 5 to 0 on new lines.

For loops are useful when you want to iterate a number of times, it is also great when iterating through strings or arrays and their indexes can be accessed at the ith location.

for (int i = 1; i <= 10; i++) {
  cout << i << endl;
}

// Outputs numbers 1 to 10 on new lines.
Pointers

When creating a variable, it is stored in memory and to get a reference to that variable, you can use the '&' operator to get the address.

var = 16;
varRef = &var; // Memory address of var
x = var;
varVal = *varRef; // Pointer to the variable stored in memory, equal to 16
Functions

In C++, funcitons are declared with a type and a name, parameters if needed, and then the statement. Functions of type void, don't need to return anything and neither does main() but anything else, for example, a bool type function would have to return a bool, an int type function would have to return an int and so on.

bool canRun(int number) {
  if (number == 5)
    return true;

  return false;
}
